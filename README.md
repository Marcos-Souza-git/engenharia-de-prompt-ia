# Comparação de Prompts em IA

##Prompt Genérico
Pergunta: O que é uma Machine Learning?

Resposta:
Machine Learning é um subcampo da Inteligência Artificial (IA) dedicado ao desenvolvimento de métodos e algoritmos que permitem que computadores aprendam a partir de dados e aprimorem seu desempenho ao longo do tempo, sem a necessidade de programação explícita para cada tarefa específica.

##Prompt Estruturado

Prompt:
Atue como um professor. Explique machine learning para um iniciante, usando analógia com esportes.

Resposta:

Para um iniciante, podemos pensar nisso como um processo de treinamento: o computador recebe muitos exemplos, analisa o que acontece em cada caso e começa a identificar padrões que ajudam a tomar decisões ou fazer previsões no futuro. Uma analogia com esportes ajuda a visualizar melhor. Imagine um técnico que analisa vários jogos de futebol para entender quais jogadas aumentam a chance de marcar gols. Com o tempo e com mais partidas observadas, ele consegue orientar melhor o time e prever situações do jogo. No machine learning acontece algo semelhante: o algoritmo “treina” com muitos dados, aprende com os resultados e melhora seu desempenho, assim como um time que evolui depois de muitos treinos e partidas.

##Análise Crítica 

A resposta do prompt estrutura foi mais útil e facil de entender, pois nela trouxe mais detalhes de como funciona machine learning unsando uma analógia de esportes.
